Game Title: Iridos
Short description of game: Iridos is a puzzle-platformer, where the player must climb to the top of a dangerous mountain
controls: 
W/Space-bar to jump
A to alk left
D to walk right
Left-Click to throw pear if in posestion
E/Left-Click to pick up spear if thrown.
Left-Shift to dash
Installation Instructions: unzip downloaded folder including other files than the executable.
